  OS running 
  RStudio running, clean workspace/history, standard window set-up
    (if some windows are not shown, RStudio state can be ensured with Ctrl + Alt + Shift + 0)

Item 1: getting and setting working directory

(a) *manually* type the following command into the console, followed by Enter:
<commands start>
getwd()
<commands end>

Outcome: 
In the console, "[1]" followed by a string representation of the use root directory is printed

(b) *manually* type the following command into the console, followed by Enter:
<commands start>
setwd("[outfilepath]")
<commands end>
(where filepath is replaced by string representation of file path to test, use slash "/" only, not backslash "\")

Outcome:
new console prompt ">" appears on console

Item 3: saving and loading scripts

(a) Click "File" menu -> "New File" -> "R Script"
Outcome: opens an empty script window with a tab called "untitled"

(b) *manually* type the following text into the script window:
<text starts>
supercalifragilistic
<text ends>
Outcome: the text should be shown in the script window

(c) Click "File" menu -> "Save As...", file/path selection window opens
In file/path selection window:
Navigate to [outfilepath], enter filename "testscript.R" in the "File Name" text field, click "Save"
Click "File" menu -> "Save As..."
Navigate to [outfilepath], enter filename "testscript2.R" in the "File Name" text field, click "Save"

Outcome:
two files are created in [outfilepath], named "testscript.R" and "testscript2.R"
The script window has a *single* tab open, called testscript2.R

(d) Click "File" menu -> "Open File..."
Navigate to [filepath] and select/confirm "testscript.R"

Outcome:
a new tab named "testscript.R" opens in the upper left script window
In the window is shown a single line of text, "supercalifragilistic"
